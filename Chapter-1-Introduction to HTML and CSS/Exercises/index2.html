HTML & CSS Quick Reference Guide

A concise guide to essential HTML tags and CSS properties.

Essential HTML Tags

<h1> to <h6> - Headings

Defines headings with varying importance.

<h1>Main Heading</h1>

<p> - Paragraph

Defines a block of text.

<p>This is a paragraph.</p>

<a> - Anchor

Creates hyperlinks.

<a href="https://example.com">Visit</a>

<img> - Image

Embeds images.

<img src="image.jpg" alt="Image">

<ul> & <ol> - Lists

Creates lists.

<ul><li>Item</li></ul>

<table> - Table

Displays data.

<table><tr><td>Data</td></tr></table>

<form> - Form

Creates user input fields.

<form><input type="text"></form>

<button> - Button

Defines clickable buttons.

<button>Click me</button>

<div> - Division

Groups elements.

<div>Content</div>

<span> - Inline Container

Styles inline elements.

<span>Text</span>

Essential CSS Properties

color

Sets text color.

p { color: grey; }

background-color

Sets background.

body { background-color: black; }

font-family

Defines text style.

p { font-family: Arial; }

border

Adds element borders.

div { border: 1px solid white; }

padding

Controls inner spacing.

div { padding: 10px; }

margin

Controls outer spacing.

div { margin: 10px; }

width

Sets width.

div { width: 50%; }

height

Sets height.

div { height: 100px; }

text-align

Aligns text.

p { text-align: center; }

display

Defines element layout.

div { display: flex; }
